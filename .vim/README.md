# vim config file
## TO INSTALL VIM/DEIN.VIM

`$ curl https://raw.githubusercontent.com/Shougo/dein.vim/master/bin/installer.sh > installer.sh`

`$ sh ./installer.sh ~/.vim/bundles`

`$ vim`

## LEADER : Space

## vim-fugitive  
git add -p : L-ga  
git commit : L-gc  
git status : L-gs  
git push   : L-gp  
git diff   : L-gd  
git log    : L-gl  

## nerdtree
NerdTreeToggle : L-nt  
open file(folder)  : o  
close directory : x  
open file remaining cursor on tree : go  
open with tabs : t  
open with tabs without moves : T  
open with vertical split : s  
open with vertical split witout moves : gs  
move parent directory : p  

## vim-commentary
comment out a line : gcc  
comment out lines  : gc  

## vim-airline
open new file : :e [filename]  
move prev tab : C-l  
move next tab : C-h
